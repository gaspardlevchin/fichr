# Product vision

Fichr is not just a CSV importer. CSV is the first structured entry point for
building reliable product sheets.

The product direction is intake -> organization -> supervision -> deliverables.
Users should be able to bring together product, collection, or project
materials; Fichr helps organize them, connect related information, highlight
what is missing, propose under supervision, and produce clean deliverables.

AI proposes. The user decides. AI must never silently modify validated data,
never invent missing facts, and never create fake success states.

Future work can extend intake to files, links, images, collections, and richer
deliverables, while keeping the same principle: controlled software actions,
clear human review, and no magic black-box automation.
