# Roadmap

Fichr should grow from intake to organization, supervision, and deliverables.
Do not reduce the product to CSV import, and do not jump directly to
multi-file AI. Build the workflow progressively, keep the user as decision
maker, and avoid fake automation.

1. Stabilize AI diagnostics and usage limits.
2. Harden private beta auth and workspace access.
3. Improve `/products/[productId]` as the V1 product hub.
4. Add demo mode with sample data.
5. Improve import onboarding.
6. Add incoming assets / file intake foundation.
7. Add manual links between files and products.
8. Add collections and projects.
9. Add intelligent link suggestions.
10. Add advanced deliverables: atelier dossier, collection dossier, exportable pack.
11. Add beta landing page.
12. Add private hosting.
13. Add plans, quotas, and Stripe later.
14. Add Notion.
15. Add Shopify after strong validation.
